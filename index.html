Desenvolvimento Web

HTML

O HyperText Markup Language (Linguagem de Marcação de Hipertexto), ou simplesmente HTML, é uma linguagem de marcação que pode ser interpretada pelos navegadores, sendo assim, utilizada para a criação de páginas na web. No Desenvolvimento web o HTML é utilizado para definir a estrutura da página. Atualmente o HTML se encontra em sua quinta versão (HTML 5) que apresentou diversos recursos para facilitar a implementação de diversos tipos de conteúdos nas páginas web. Um dos principais diferenciais do HTML é sua capacidade de ser facilmente legível por seres humanos, ainda que nunca tenham tido contato.

CSS

Cascading Style Sheets (Folha de Estilo em Cascatas), ou simplesmente CSS, é uma linguagem utilizada como mecanismo para adicionar estilos às páginas web, ou seja, é utilizada em conjunto com o HTML para melhorar a experiência visual do usuário com o conteúdo que está sendo exibido. No entanto, não subestime o CSS, apesar de com essa definição a linguagem aparentar ser um tanto quanto simples e limitada, a diferença que faz um folha de estilo bem escrita normalmente é crucial para fazer com que o conteúdo da página seja realmente apresentado de uma forma que cumpra o objetivo pelo qual foi colocado.

JavaScript

O JavaScript, ou também abreviadamente JS, é uma linguagem de programação estruturada interpretada. Enquanto o HTML e o CSS são responsáveis pelo conteúdo apresentado, ela envolve a lógica de programação, sendo utilizado inclusive para manipular elementos e a própria estilização das páginas web, conforme o objetivo dos desenvolvedores. Dentre as utilizadas no Desenvolvimento web, por ser mais próxima da linguagem de máquina (ainda que esteja longe disso), é a menos legível pelos humanos, tendo uma estrutura muitas vezes assustadora para que, nunca teve contato com uma linguagem de programação anteriormente.
